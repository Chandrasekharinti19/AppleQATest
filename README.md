# AppleQATest

A SwiftUI Apple Store demo app with production-style XCTest UI automation.

## Features
- Product variants, colors, and storage options
- Cart and checkout flow
- Shipping and payment validation
- Apple-style image gallery
- Promo code support

## Test Automation
- XCTest UI tests
- Page Object Model
- Product configuration matrix test
- Xcode Test Plans
- DEV / QA / STAGING environments
- Parallel execution
- Screenshot capture on failure
